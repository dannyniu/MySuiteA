MySuiteA

MySuiteA is a collection of cryptographic algorithms implemented in the C programming language. All codes (except testing codes) are implemented in free-standing C, without any call to system libraries, and can be used in embedded systems. All symmetric-key algorithms can be instantiated at both compile-time and run-time.

For a list of algorithms implemented in the suite, see [Algorithms] For a information on the design of API, see [API] .
